Implementation of RDBMS for Homer Consulting

Tech Stack: Oracle 11g, SQL, Visio

Homer consulting is a firm that is looking for the set up of a completely new Database based on Oracle 11g. They also have some complex SQL questions that they need help building queries for. The ER diagram they want to implement is as below. I created tables with the necessary contraints such as appropriate primary keys, foriegn keys and several other contraints that check dates, uniques etc.

Once the database was created using Oracle 11g, It was time to implement some tough SQL questions in the form of queries.

  • The company’s human resources department needs information on the current training levels of the employees. They need information that shows, for each employee, and his/her acquired skills, the number of times he or she received training for that skill, the most recent date of the training, and the number of months (full months only) that have passed since the most recent training for the skill. Include all employees in the output, even if they have not received any training yet.

    SELECT EMP_NUM, EMP_FIRST||' '||EMP_LAST AS "EMPLOYEE", SKILL.DESCRIPTION, COUNT(CODE) AS "NO_OF_TIMES", NVL((TO_CHAR(TRUNC(((SYSDATE) - MAX(TRAINING.DATE_ACQUIRED))/30))),'N/A') AS "MONTHS",NVL(TO_CHAR(MAX(TRAINING.DATE_ACQUIRED)),'N/A') AS "RECENT DATE"
FROM EMPLOYEE LEFT OUTER JOIN TRAINING USING(EMP_NUM)
LEFT OUTER JOIN SKILL USING (CODE)
GROUP BY EMP_NUM,EMP_FIRST,EMP_LAST,SKILL.DESCRIPTION;

  • Due to a recent re-organization, the company needs a list that shows who supervises who. List all the employees in the company clearly indicating the organizational hierarchy. Include the “level” of the hierarchy each employee is at and the department name of the employee.

    SELECT LEVEL,
LPAD(' ', 3*(LEVEL - 1)) || EMP_NUM || ' ' ||
EMP_FIRST || ' ' || EMP_LAST AS "EMPLOYEE", DEPT_CODE,DEPARTMENT.NAME
FROM EMPLOYEE JOIN DEPARTMENT USING(DEPT_CODE)
START WITH EMP_NUM = '400'
CONNECT BY PRIOR EMP_NUM = SUPER_ID;
LEFT OUTER JOIN SKILL USING (CODE)
GROUP BY EMP_NUM,EMP_FIRST,EMP_LAST,SKILL.DESCRIPTION;

SELECT A.EMP_FIRST||' '||A.EMP_LAST AS "EMPLOYEE", EMP_NUM2,NVL(TO_CHAR(TRAIN_ID),'N/A') AS "TRAIN_ID",NVL(TO_CHAR(DATE2),'N/A') AS DATE2,TRUNC((SYSDATE-DATE2)/30) AS "MONTHS",NAME,C.EMP_NUM AS "SUPER_ID",C.EMP_FIRST||' '||C.EMP_LAST AS "SUPERVISOR" FROM EMPLOYEE A FULL OUTER JOIN(SELECT B.EMP_NUM AS "EMP_NUM2",T.NAME AS "NAME",T.TRAIN_NUM AS "TRAIN_ID",T.DATE_ACQUIRED AS "DATE2" FROM EMPLOYEE B LEFT OUTER JOIN TRAINING T ON(B.EMP_NUM=T.EMP_NUM)

LEFT OUTER JOIN EMPLOYEE C ON (A.SUPER_ID = C.EMP_NUM)
ORDER BY A.EMP_NUM,NAME,DATE2;

  • For every ongoing project (i.e., a project with total cost being NULL) provide the project name, project start date. In addition, for each month of the project list the number of employees assigned that month and hours spent on the project that month. At the end provide an overall total of the number of employees assigned to the project and the total hours spent on the project.

    SELECT NVL(TO_CHAR(PROJ_NUMBER),'TOTAL') AS "PROJ_NUMBER", NAME, START_DATE, (EXTRACT(MONTH FROM DATE_ASSIGNED)||' '||EXTRACT(YEAR FROM DATE_ASSIGNED)) AS "MONTH", COUNT(EMP_NUM) AS "NO_OF_ASSIGNMENTS_PER_MONTH", SUM(HOURS_USED),COUNT(DISTINCT EMP_NUM) AS "UNIQUE_EMPLOYEES"
FROM PROJECT JOIN ASSIGNMENT USING(PROJ_NUMBER)
JOIN EMPLOYEE USING (EMP_NUM)
WHERE TOTAL_COST IS NULL
GROUP BY GROUPING SETS ((NAME, START_DATE,PROJ_NUMBER,DATE_ASSIGNED),(PROJ_NUMBER,NAME),());

  • The company has decided to adopt a bonus policy for their employees. Each employee will receive a bonus of $200 for every project started last calendar year, on which they have charged at least 40 hours. As a result, management has asked you to update the employee records to include their current bonus information. In order to accomplish this, you will have to add another column to the EMPLOYEE table (i.e., column BONUS_AMT), before you update the data. After the single Update statement is performed, list the contents of the Employee table.

    SQL> UPDATE EMPLOYEE
SET EMPLOYEE.BONUS_AMT = ( SELECT B.BONUS_AMT
                  FROM (SELECT EMP_NUM, EMP_FIRST||' '||EMP_LAST AS "EMPLOYEE", COUNT(PROJ_NUMBER) AS "NO_OF_PROJECTS", (200*COUNT(PROJ_NUMBER)) AS BONUS_AMT, SUM(HOURS_USED) AS "HOURS"
FROM EMPLOYEE JOIN ASSIGNMENT USING (EMP_NUM)
JOIN PROJECT USING (PROJ_NUMBER)
WHERE EXTRACT(YEAR FROM START_DATE) IN (EXTRACT (YEAR FROM SYSDATE)-1)
GROUP BY EMP_NUM, EMP_FIRST,EMP_LAST
HAVING SUM(HOURS_USED) >=40) B
WHERE EMPLOYEE.EMP_NUM = B.EMP_NUM);
